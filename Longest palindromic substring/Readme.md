# Longest palindromic substring

Given a string s, find the longest palindromic substring in s. You may assume that the maximum length of s is 1000.<br>

Example 1:

Input: "babad"<br>
Output: "bab"<br>
Note: "aba" is also a valid answer.<br>

Example 2:

Input: "cbbd"<br>
Output: "bb"
